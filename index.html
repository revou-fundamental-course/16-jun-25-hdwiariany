Kalkulator Segitiga

Hitung semua jenis yang berkaitan dengan segitiga secara cepat dan mudah!

Luas Segitiga

Rumus:
L = ½ × alas × tinggi

Hitung Tinggi

Rumus:
T = (2 × luas) / alas

Hitung Alas

Rumus:
A = (2 × luas) / tinggi

Hitung Sisi Miring (Siku-siku)

Rumus:
c = √(alas² + tinggi²)

Keliling Segitiga

Rumus:
K = sisi A + sisi B + sisi C

Hitung Sisi Miring (Segitiga Sama Kaki)

Rumus:
sisi miring = (keliling - alas) / 2

Luas Segitiga Sembarang

Rumus:
L = √(s(s - a)(s - b)(s - c)),
dengan s = (a + b + c) / 2

Hitung Sudut Segitiga

Rumus:
cos A = (b² + c² - a²) / 2bc
cos B = (a² + c² - b²) / 2ac
cos C = (a² + b² - c²) / 2ab
